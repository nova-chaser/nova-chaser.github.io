1、什么是集成学习

集成学习通过建立几个模型来解决单一预测问题。工作原理是生成多个分类器/模型。各自独立的学习与做出预测。这些预测最后结合成组合预测，因此优于任何⼀个单分类的做出预测。

2.2 随机森林构造过程

机器学习中，随机森林是一个包含多个决策树的分类器，并其输出的类别是由个别树输出的类别的众数而定。

随机森林 = Bagging + 决策树

例如：训练了5个树，其中有4个树的结果是TRUE，1个树的结果是FALSE，那么最终投票结果就是TRUE

随机森林构造的关键步骤：
（1）一次随机选出一个样本，有放回的抽样，重复N次

（2）随机选出m个特征m<<M，建立决策树（M表示特征数目）

  • 为什么要随机抽样训练集？

如果不进行随机抽样，每棵树的训练集都一样，那么最终训练出的树分类结果也完全是一样的

  • 为什么要有放回地抽样？

如果不是有放回的抽样，那么每课树的训练样本都是不同的，都是没有交集的，这样每棵树都是“有偏的”，也就是每棵树训练出来都是有很大的差异的；而随机森林最后分类取决于多棵树（弱分类器）的投票表决

2.3 包外估计（Out-Of-Bag Estimate）

包外估计可用于：

  • 基学习器是神经网络时，可使用包外样本来辅助早期停止以减少过拟合

3、随机森林API

sklearn.ensemble.RandomForestClassifier(n_estimators=10, criterion=’gini’, max_depth=None, bootstrap=True, random_state=None, min_samples_split=2)

  • n_estimators：integer，optional(default = 10)森林里面的树的数量
  • criterion：string，optional（default=“gini”）分割特征的测量方法
  • max_depth：integer或None，optional(default = None)树的最大深度
  • max_features="auto”,每个决策树的最⼤特征数量
    • If “auto”，then max_features=sqrt(n_features)
    • If “sqrt”，then max_features=sqrt(n_features)(same as “auto”)
    • If “log2”，then max_features=log2(n_features)
    • If None，then max_features=n_features
  • bootstrap： boolean，optional(default =True)，是否使用放回取样
  • min_samples_split：内部节点再划分所需要最小样本数
  • min_samples_leaf：叶子结点最小的样本数

5、Boosting

随着学习的积累从弱到强

简而言之：每新加入⼀个弱学习器，整体能⼒就会得到提升

代表算法：Adaboost，GBDT，XGBoost，LightGBM

5.2 Bagging集成与boosting集成的区别：

  • 数据方面：
    • Bagging：对数据进行采样训练
    • Boosting：根据前一轮学习结果调整数据的重要性
  • 投票方面：
    • Bagging：所有学习器平权投票
    • Boosting：对学习器加权投票
  • 学习顺序：
    • Bagging：并行，每个学习器没有依赖关系
    • Boosting：串行，有先后顺序
  • 主要作用：
    • Bagging：提高泛化性能（解决过拟合，降低方差）
    • Boosting：提高训练精度（解决欠拟合，降低偏差）

5.3 AdaBoost

1、构造过程的细节

  1. 初始化训练数据权重相等，训练第一个学习器
    1. 假设每个训练样本在基分类器的学习中作用相同，这一假设可以保证第一步能在原始数据上学习基本分类器H1(x)
  2. AdaBoost反复学习基本分类器，在每一轮m = 1,2,…,M顺次执行下列操作：
    1. 在权值分布为Dt的训练数据上，确定基分类器
    2. 计算该学习器在训练数据中的错误率
    3. 计算该学习器的投票权重
    4. 根据投票权重，对训练数据重新赋权Zt为归一化因子（规范化常数）
      • $ α_t $是分配给弱分类器的权重。
      • $ y_i $是样本$ x_i $的真实标签。
      • $ h_t(x_i) $是弱分类器对样本$ x_i $的预测结果。
    5. 重复执行a~d，M次。
  3. 对M个学习器进行加权投票

6、GBDT

GBDT 的全称是 Gradient Boosting Decision Tree，梯度提升树，在传统机器学习算法中，GBDT算的上TOP3的算法。 想要理解GBDT的真正意义，那就必须理解GBDT中的Gradient Boosting 和Decision Tree分别是什么

GBDT使用的是CART回归树

为什么不使用CART分类树？因为GBDT每次迭代要拟合的是梯度值，是连续值所以要用回归树

6.1 Gradient Boosting：拟合负梯度

梯度提升树（Gradient Boosting）是提升树（Boosting Tree）的一种改进算法。

对于提升树，有一个通俗的例子：假如有个人30岁，我们先先用20岁去拟合，发现损失有10岁，这时我们用6岁去拟合剩下的损失，发现差距还有4岁，第三轮我们用3岁拟合剩下的差距，差距就只有一岁了。如果我们的迭代轮数还没有完，可以继续迭代下面，每一轮迭代，拟合的岁数误差都会减小。最后将每次拟合的岁数加起来便是模型输出的结果。

提升树算法：

  1. 初始化$ f_0(x) = 0 $
  2. 对m=1,2，…，M
    1. 计算残差$ r_{mi}=y_i-f_{m-1}(x),i=1,2,…,N $
    2. 拟合残差$ r_{mi} $学习一个回归树，得到$ h_m(x)
      $
    3. 更新$ f_m(x)=f_{m-1}+h_m(x) $
  3. 得到回归问题提升树$ f_M(x)=\sum_{m=1}^M h_m(x) $
