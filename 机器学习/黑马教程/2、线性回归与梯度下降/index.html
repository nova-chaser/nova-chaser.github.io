2、常见的梯度下降算法

2.1 全梯度下降算法（ Batch Gradient Descent）

是梯度下降算法的常用形式，具体做法是在更新参数时使用所有的样本来进行更新。计算训练集所有样本误差，对其求和再取平均值作为目标函数。是在整个训练数据集上计算损失函数关于参数θ的梯度。

注意：

  • 因为在执行每次更新时，我们需要在整个数据集上计算所有的梯度，所以批梯度下降法的速度会很慢，同时，批梯度下降法⽆法处理超出内存容量限制的数据集。
  • 全梯度下降同样也不能在线更新模型，即在运行的过程中，不能增加新的样本

2.2 随机梯度下降算法（Stochastic Gradient Descent, SGD）

由于全梯度下降每迭代更新⼀次权重都需要计算所有样本误差，而实际问题中经常有上亿的训练样本，故效率偏低，且容易陷入局部最优解，因此提出了随机梯度下降算法。

即每次只代入计算一个样本目标函数的梯度来更新权重，再取下一个样本重复此过程，直到损失函数拟合。

此过程简单，高效，通常可以较好地避免更新迭代收敛到局部最优解。

2.3 小批量梯度下降算法（Mini-batch Gradient Descent）

每次从训练集中随机抽取一个小样本集，在此之上使用全梯度下降。

小样本集所含样本个数通常为batch_size，通常为2的幂次方。

2.4 随机平均梯度下降算法（Stochastic Average Gradient Descent，SAG）

在SGD中， 虽然避开了运算成本大的问题，但对于大数据训练而言，SG效果常不尽如⼈意，因为每⼀轮梯度更新都完全与上⼀轮的数据和梯度无关。

随机平均梯度算法克服了这个问题，在内存中为每一个梯度都维护了一个旧梯度，随机选择第i个样本来更新此样本的梯度，其他样本的梯度保持不变，然后求得所有梯度的平均值，进而更新参数。

  • SGD是当前权重减步长乘梯度，得到新的权重。而SAG，在第K步迭代时，考虑这一步与前n-1个梯度的平均值，当前权重减去步长乘最近n个梯度的平均值。
  • n是自己设置的，n=1时，就是普通的SGD
  • 其成本远小于mini-batch SGD，和SGD相当。收敛速度相对于SGD快很多。

3、线性回归API再介绍

sklearn.linear_model.LinearRegression(fit_intercept=True)

  • 通过正规方程优化
  • 参数
    • fit_intercept：是否计算偏置
  • 属性
    • LinearRegression.coef_：回归系数（即参数是多少）
    • LinearRegression.intercept_：偏置（偏置值）

sklearn.linear_model.SGDRegressor(loss="squared_loss", fit_intercept=True, learning_rate='invscaling', eta0=0.01)

  • 实现了随机梯度下降，支持不同的loss函数和正则化惩罚项来拟合线性回归模型
  • 参数
    • loss：损失类型
      • loss=“squared_loss”：普通最小二乘法
    • fit_intercept：是否计算偏置
    • learning_rate：string，optional
      • 学习率填充
      • ‘constant’：eta = eta0
      • ‘optimal’： eta = 1.0/(alpha*(t+t0))[default]
      • invscaling： eta =eta0/pow(t,power_t)
        • power_t=0.25：存在⽗类当中
      • 对于⼀个常数值的学习率来说，可以使⽤learning_rate=‘constant’，并使⽤eta0来指定学习率。
    • 属性
      • LinearRegression.coef_：回归系数（即参数是多少）
      • LinearRegression.intercept_：偏置（偏置值）

5、过拟合的解决方案：正则化

在学习的时候，数据提供的特征有些影响模型复杂度或者这个特征的数据点异常较多，所以算法在学习时尽量减少这个特征的影响（甚至删除），这就是正则化

5.1 类别

  • L2正则化
    • 作用：可以使得其中一些W的都很小，都接近于0，削弱某个特征的影响
    • 优点：越小的参数说明模型越简单，越简单的模型则越不容易产生过拟合现象
    • Ridge回归
  • L1正则化
    • 作用：可以使得其中一些W值直接为0，删除这个特征影响
    • LASSO回归

5.2 Ridge Regression（岭回归，又名Tikhonov regularization）

岭回归是线性回归的正则化版本，正则项：，公式为：

α=0：岭回归退化为线性回归

sklearn.linear_model.Ridge(alpha=1.0,fit_intercept=True,solver="auto",normalize=False)

  • 具有L2正则化的线性回归
  • alpha：正则化力度，也叫λ
    • λ取值：0~1 1~10
  • solver：会根据数据自动选择优化算法
    • sag：如果数据集，特征都比较大，选择该随机梯度下降优化
  • normalize：数据是否可以进行标准化
    • normalize=False:可以在fit之前调⽤preprocessing.StandardScaler标准化数据
  • Ridge.coef_：回归权重
  • Ridge.intercept_：回归偏置

Ridge⽅法相当于SGDRegressor(penalty='l2', loss="squared_loss"),只不过SGDRegressor实现了⼀个普通的随机梯度下降学习，推荐使⽤Ridge(实现了SAG)

sklearn.linear_model.RidgeCV(_BaseRidgeCV, RegressorMixin)

  • 具有l2正则化的线性回归，可以进⾏交叉验证
  • coef_：回归系数

5.4 Elastic Net（弹性网络）

弹性⽹络在岭回归和Lasso回归中进⾏了折中，通过 混合⽐(mix ratio)r进⾏控制：

  • r=0：弹性网络变为岭回归
  • r=1：弹性网络变为Lasso回归
